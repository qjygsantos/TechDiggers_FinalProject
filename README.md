# TeamTechDiggersProject

#### Clone this repository

```
git clone https://github.com/qhagayla/TeamTechDiggersProject.git
```
### Frontend
#### Install dependencies

```
npm install
```

### Usage

#### Run the app

```
npm expo start
```

### Backend
#### Run the server in four (4) additional terminals
```
node loginserver.js
```
```
node signupserver.js
```
```
node meserver.js
```
```
python server.py
